# 🎓 Semester 3 (2024/2025 Session 1)

Semester 3 focuses on strengthening core Computer Science and Data Engineering knowledge through advanced topics in data structures, database systems, software engineering principles, and computer networking. This semester emphasizes algorithmic thinking, system design, database modeling, and real-world problem solving through both individual and group-based projects.

---

## 📚 Courses Taken

### SECJ2013 – Data Structures and Algorithms

This course emphasizes the concepts of data structures and algorithm design, focusing on efficient problem-solving techniques. Students learn how to implement and analyze various data structures such as linked lists, stacks, queues, and trees, along with sorting and searching algorithms. The course also introduces recursion and algorithm complexity analysis using Big-O notation.

#### Key Topics Learned

* Algorithm analysis and Big-O notation
* Recursion techniques
* Sorting algorithms (Bubble, Selection, Insertion, Merge, Quick Sort)
* Searching algorithms (Linear and Binary Search)
* Linked Lists (Singly, Doubly, Circular)
* Stack and Queue implementations
* Trees and Binary Search Trees

#### Skills Acquired

* Algorithm design and optimization
* Problem-solving using data structures
* C++ programming implementation
* Complexity analysis
* Logical and computational thinking

#### Technologies Used

* C++
* Visual Studio Code / Dev C++

---

### SECP2523 – Database

This course introduces database system concepts including relational database design, SQL programming, normalization, and ER diagram modeling. Students are trained to design efficient database systems and implement SQL queries using DBMS tools such as MySQL.

#### Key Topics Learned

* Database system concepts and architecture
* Entity Relationship Diagram (ERD)
* Normalization (1NF, 2NF, 3NF)
* SQL (DDL, DML, DCL)
* Joins, Subqueries, Aggregation
* Database design lifecycle
* MySQL implementation

#### Skills Acquired

* Database design and modeling
* SQL query development
* Data normalization techniques
* Logical database structuring
* DBMS usage and implementation

#### Technologies Used

* MySQL
* phpMyAdmin
* DBMS tools

---

### SECP3204 – Software Engineering

This course provides an introduction to software engineering principles and systematic software development processes. It covers requirement analysis, system design, UML modeling, and software testing techniques. Students work in teams to develop a complete software project using structured engineering approaches.

#### Key Topics Learned

* Software development life cycle (SDLC)
* Agile and Waterfall models
* Requirement engineering (SRS)
* UML diagrams (Use Case, Class, Sequence)
* Software architecture design
* Software testing and validation

#### Skills Acquired

* Software system design
* UML modeling and documentation
* Team-based software development
* Requirement analysis
* Project management fundamentals

#### Technologies Used

* UML tools (draw.io)

---

### SECR1213 – Network Communications

This course covers fundamental concepts of computer networking based on the TCP/IP model. Students learn how data is transmitted across networks, including routing, IP addressing, transport protocols, and wireless communication technologies. Practical labs involve tools such as Wireshark and Packet Tracer.

#### Key Topics Learned

* OSI and TCP/IP models
* Application, Transport, and Network layers
* IP addressing and subnetting
* Routing and switching
* LAN and wireless networks
* Network troubleshooting tools

#### Skills Acquired

* Network design and analysis
* Packet tracing and troubleshooting
* Subnetting and IP configuration
* Network simulation skills
* Communication protocol understanding

#### Technologies Used

* Cisco Packet Tracer
* Wireshark

---

### ULRF2832 – Swimming (Women)

This co-curricular course focuses on physical fitness, water safety, and swimming techniques. Students learn basic swimming strokes, breathing techniques, and endurance training while engaging in experiential learning activities.

#### Key Topics Learned

* Basic swimming techniques
* Water safety and survival skills
* Stroke development (freestyle, breaststroke, etc.)
* Physical conditioning in water
* Team-based experiential learning activities

#### Skills Acquired

* Physical fitness and endurance
* Water survival skills
* Discipline and teamwork
* Self-confidence in aquatic environments
