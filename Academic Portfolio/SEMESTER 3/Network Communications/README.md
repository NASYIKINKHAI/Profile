### SECR1213 – Network Communications

This course covers fundamental concepts of computer networking based on the TCP/IP model. Students learn how data is transmitted across networks, including routing, IP addressing, transport protocols, and wireless communication technologies. Practical labs involve tools such as Wireshark and Packet Tracer.

#### Key Topics Learned

* OSI and TCP/IP models
* Application, Transport, and Network layers
* IP addressing and subnetting
* Routing and switching
* LAN and wireless networks
* Network troubleshooting tools

#### Skills Acquired

* Network design and analysis
* Packet tracing and troubleshooting
* Subnetting and IP configuration
* Network simulation skills
* Communication protocol understanding

#### Technologies Used

* Cisco Packet Tracer
* Wireshark
