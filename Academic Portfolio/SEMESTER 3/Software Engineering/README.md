### SECP3204 – Software Engineering

This course provides an introduction to software engineering principles and systematic software development processes. It covers requirement analysis, system design, UML modeling, and software testing techniques. Students work in teams to develop a complete software project using structured engineering approaches.

#### Key Topics Learned

* Software development life cycle (SDLC)
* Agile and Waterfall models
* Requirement engineering (SRS)
* UML diagrams (Use Case, Class, Sequence)
* Software architecture design
* Software testing and validation

#### Skills Acquired

* Software system design
* UML modeling and documentation
* Team-based software development
* Requirement analysis
* Project management fundamentals

#### Technologies Used

* UML tools (draw.io)
