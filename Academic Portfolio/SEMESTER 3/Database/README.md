### SECP2523 – Database

This course introduces database system concepts including relational database design, SQL programming, normalization, and ER diagram modeling. Students are trained to design efficient database systems and implement SQL queries using DBMS tools such as MySQL.

#### Key Topics Learned

* Database system concepts and architecture
* Entity Relationship Diagram (ERD)
* Normalization (1NF, 2NF, 3NF)
* SQL (DDL, DML, DCL)
* Joins, Subqueries, Aggregation
* Database design lifecycle
* MySQL implementation

#### Skills Acquired

* Database design and modeling
* SQL query development
* Data normalization techniques
* Logical database structuring
* DBMS usage and implementation

#### Technologies Used

* MySQL
* phpMyAdmin
