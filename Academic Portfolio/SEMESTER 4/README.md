# 🎓 Semester 4 (2024/2025 Session 2)

Semester 4 focuses on strengthening advanced programming, systems understanding, data analysis techniques, communication skills, and critical thinking. The courses taken in this semester emphasize object-oriented programming, operating systems, data analytics programming, data mining, professional communication, and philosophical reasoning, preparing students for more advanced computing concepts and real-world applications.

## 📚 Courses Taken

---

### SECJ2154 – Object Oriented Programming

This course presents the concepts of object orientation and object-oriented programming techniques using the Java programming language. It provides students with a thorough understanding of basic Java constructs such as data types and operations. The course also covers exception handling and file input/output.

#### Key Topics Learned

* Object-Oriented Concepts
* Classes and Objects
* Encapsulation and Constructors
* Inheritance and Polymorphism
* Abstract Classes and Interfaces
* Exception Handling
* File Input/Output

#### Skills Acquired

* Java Programming
* Object-Oriented Design
* Problem Solving using OOP
* Software Development Skills
* Debugging and Exception Handling

---

### SECR2043 – Operating Systems

This course covers the fundamental concepts of operating systems, including process management, concurrency, memory management, file systems, and storage management. It provides understanding of how operating systems manage hardware resources and execute processes efficiently.

#### Key Topics Learned

* Operating System Concepts
* Process Management
* CPU Scheduling Algorithms
* Process Synchronization (Mutex, Semaphores)
* Deadlocks
* Memory Management (Paging and Segmentation)
* Virtual Memory
* File Systems
* Storage Management

#### Skills Acquired

* System-Level Understanding
* Process Scheduling Analysis
* Synchronization and Concurrency Control
* Memory Allocation Concepts
* Analytical Problem Solving

---
