### SCSP3744 – Enterprise System Design and Modeling

This course focuses on enterprise information systems, system integration, and enterprise architecture frameworks. It emphasizes real-world business process modeling and enterprise system design.

#### Key Topics Learned
- Enterprise Resource Planning (ERP)
- Enterprise Architecture (Zachman & TOGAF)
- Business Process Design
- System Implementation Strategies

#### Skills Acquired
- Enterprise System Analysis
- System Design & Architecture Modeling
- Business Process Understanding
- Problem Solving in Enterprise Systems
- Team Collaboration & Presentation Skills
