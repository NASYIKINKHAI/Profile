### UHLC1112 – Mandarin Language I

This introductory language course builds basic Mandarin communication skills, focusing on speaking, reading, and writing fundamentals for everyday and academic use.

#### Key Topics Learned
- Basic Mandarin Grammar
- Vocabulary Development
- Sentence Construction
- Pronunciation & Tones
- Basic Conversation Skills

#### Skills Acquired
- Basic Mandarin Communication
- Listening & Speaking Skills
- Language Learning Skills
