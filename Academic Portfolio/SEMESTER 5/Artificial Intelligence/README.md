### SECJ3553 – Artificial Intelligence

This foundational course introduces students to Artificial Intelligence concepts, including search algorithms, knowledge representation, and machine learning techniques. It combines theoretical understanding with practical applications in real-world AI problems.

#### Key Topics Learned
- Search Algorithms (DFS, BFS, A*)
- Knowledge Representation
- Problem Solving Techniques
- Game Playing (Minimax, Alpha-Beta Pruning)
- Machine Learning Fundamentals
- Natural Language Processing
- Computer Vision Basics

#### Skills Acquired
- AI Problem Analysis
- Algorithm Design & Implementation
- Machine Learning Fundamentals
- Logical Thinking & Reasoning
- Practical AI Application Development
