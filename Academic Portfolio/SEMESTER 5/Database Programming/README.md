### SECP3623 – Database Programming

This course enhances advanced database skills focusing on both relational (SQL) and non-relational (NoSQL) systems. Students learn scalable database design, querying techniques, and integration of database systems in applications.

#### Key Topics Learned
- Advanced SQL Queries
- Stored Procedures & Indexing
- NoSQL Databases (MongoDB)
- Key-Value & Document Stores
- Graph & Column-Family Databases
- Data Consistency Models
- Database Integration Techniques

#### Skills Acquired
- Advanced SQL Programming
- NoSQL Database Design
- Data Modeling Skills
- Database Optimization
- System Integration Skills
