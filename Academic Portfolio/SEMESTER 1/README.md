# 🎓 Semester 1 (2023/2024 Session 1)

Semester 1 established the fundamental knowledge and technical competencies required in the field of Computer Science and Data Engineering. The courses undertaken during this semester provided a strong foundation in mathematical reasoning, programming, and digital system design, while fostering analytical thinking and problem-solving skills essential for future studies and professional practice.

## 📚 Courses Taken

### SECI1013 – Discrete Structure

This course focused on the mathematical foundations of computer science, covering concepts such as set theory, logic, proof techniques, relations, functions, recurrence relations, counting methods, graph theory, trees, and finite automata. These topics are fundamental for understanding computational models, algorithms, and problem-solving techniques used in computer science.

---

### SECJ1013 – Programming Technique I

This course introduced the principles of structured programming using C++. Emphasis was placed on problem analysis, algorithm development, and program implementation through a systematic programming approach. Practical exercises and assignments strengthened the understanding of software development fundamentals and computational thinking.

#### Key Topics Learned

* Problem Solving Techniques
* Flowcharts and Pseudocode
* Variables, Data Types, and Operators
* Selection and Loop Structures
* Functions
* Arrays and Multidimensional Arrays
* File Processing
* Pointers
* Structures and User-Defined Data Types

#### Skills Acquired

* C++ Programming
* Algorithm Development
* Program Testing and Debugging
* Structured Programming
* Software Development Fundamentals

#### Technologies Used

* C++
* Visual Studio Code
* Dev C++

### SECR1013 – Digital Logic

This course provided an introduction to digital electronics and digital system design. Theoretical concepts were reinforced through laboratory activities involving logic circuit design, simulation, implementation, and verification. The course served as a foundation for understanding modern computer hardware and digital systems.

#### Key Topics Learned

* Number Systems and Codes
* Logic Gates
* Boolean Algebra
* Karnaugh Maps (K-Maps)
* Combinational Logic Circuits
* Sequential Logic Circuits
* Counters and Shift Registers
* Programmable Logic Devices (PLD)

#### Skills Acquired

* Digital Circuit Design
* Logic Simplification
* Hardware Simulation
* Digital System Analysis
* Technical Documentation and Reporting

#### Technologies Used

* DEEDS Simulator
