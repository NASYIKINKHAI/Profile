### SECJ1013 – Programming Technique I

This course introduced the principles of structured programming using C++. Emphasis was placed on problem analysis, algorithm development, and program implementation through a systematic programming approach. Practical exercises and assignments strengthened the understanding of software development fundamentals and computational thinking.

#### Key Topics Learned

* Problem Solving Techniques
* Flowcharts and Pseudocode
* Variables, Data Types, and Operators
* Selection and Loop Structures
* Functions
* Arrays and Multidimensional Arrays
* File Processing
* Pointers
* Structures and User-Defined Data Types

#### Skills Acquired

* C++ Programming
* Algorithm Development
* Program Testing and Debugging
* Structured Programming
* Software Development Fundamentals

#### Technologies Used

* C++
* Visual Studio Code
* Dev C++
