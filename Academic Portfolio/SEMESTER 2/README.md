# 🎓 Semester 2 (2023/2024 Session 2)

Semester 2 builds upon the foundational knowledge gained in Semester 1 by introducing more advanced concepts in statistical data analysis, object-oriented programming, system design, and computer architecture. This semester emphasizes practical application, analytical reasoning, and project-based learning using tools such as R programming, C++, UML modeling, and assembly language programming.

## 📚 Courses Taken

---

### SECI1143 – Probability & Statistical Data Analysis

This course introduces statistical techniques used for data analysis and interpretation. Students are exposed to descriptive statistics, probability theory, hypothesis testing, correlation, regression, and ANOVA. Practical implementation is carried out using R programming to analyze and visualize data.

#### Key Topics Learned

* Descriptive Statistics (Mean, Median, Mode, Variance, Standard Deviation)
* Data Visualization (Bar chart, Histogram, Box plot)
* Probability and Probability Distributions (Binomial, Poisson, Normal)
* Hypothesis Testing (1-sample and 2-sample tests)
* Chi-Square Test and Contingency Analysis
* Correlation Analysis (Pearson and Spearman)
* Linear Regression
* Analysis of Variance (ANOVA)

#### Skills Acquired

* Statistical Data Analysis
* Data Interpretation and Inference
* Hypothesis Testing
* Data Visualization
* Analytical and Critical Thinking

#### Technologies Used

* R Programming
* RStudio

### SECJ1023 – Programming Technique II (Object-Oriented Programming)

This course focuses on Object-Oriented Programming (OOP) concepts using C++. It emphasizes software design principles such as encapsulation, inheritance, polymorphism, and abstraction. Students develop structured and reusable programs through individual and group-based assignments and projects.

#### Key Topics Learned

* Object-Oriented Programming Principles
* Classes and Objects
* Constructors and Destructors
* Friend Functions and Operator Overloading
* Arrays of Objects and Pointers
* Inheritance and Class Hierarchies
* Polymorphism and Virtual Functions
* String Manipulation
* Templates and Exception Handling

#### Skills Acquired

* Object-Oriented Programming in C++
* Software Design and Development
* Code Reusability and Modular Design
* Debugging and Testing
* Problem Solving Using OOP

#### Technologies Used

* C++
* Visual Studio

### SECR1033 – Computer Organization and Architecture

This course provides understanding of computer system structure and hardware-level operations. It covers data representation, CPU architecture, memory systems, pipelining, and assembly language programming.

#### Key Topics Learned

* Computer System Components
* Data Representation and Arithmetic Operations
* Instruction Set Architecture (ISA)
* CPU Architecture and Instruction Cycle
* Pipelining and Performance Optimization
* Memory Hierarchy (Cache, RAM, Virtual Memory)
* Input/Output Systems
* Assembly Language Programming

#### Skills Acquired

* Low-Level Programming (Assembly Language)
* Understanding Computer Architecture
* CPU Performance Analysis
* Memory System Analysis
* Hardware-Level Problem Solving

### SECV2113 – Human-Computer Interaction

This course introduces Human-Computer Interaction (HCI) theories and user experience (UX) design principles. It focuses on designing, prototyping, and evaluating interactive systems using a user-centred approach. The course emphasizes usability, accessibility, and emotional impact in system design through iterative development and evaluation processes.


#### Key Topics Learned

* Introduction to Human-Computer Interaction (HCI)
* User Experience (UX), Usability, Accessibility, and Inclusiveness
* Human Cognition, Emotional and Social Interaction
* Interaction Design Principles
* Conceptual Models and Design Thinking
* Requirement Gathering Techniques (Interviews, Surveys, Observation)
* Task Analysis (HTA, Use Cases)
* Prototyping (Low & Medium Fidelity)
* Heuristic Evaluation and Usability Testing
* UX Evaluation and Data Analysis


#### Skills Acquired

* UX/UI Design Thinking
* User-Centered Design Approach
* Requirement Analysis and Data Gathering
* Prototyping Interactive Systems
* Usability Testing and Evaluation
* Team Collaboration in Design Projects
* Critical Thinking in Design Decisions


#### Project Highlights

* Built interactive prototypes based on real user needs
* Conducted usability testing and user evaluation
* Applied iterative UX design lifecycle
* Performed peer assessment and collaborative project development


#### Technologies / Tools Used

* Figma (UI/UX Prototyping)
* Paper Sketching / Storyboarding
* UX Evaluation Techniques

### ULRS1182-25 – Penghayatan Etika dan Peradaban (Appreciation of Ethics and Civilizations)

This course introduces ethical concepts from the perspective of different civilizations. It aims to help students understand the systems, development, progress, and cultural diversity of various societies in strengthening social cohesion. The course also discusses contemporary issues in economics, politics, social, cultural, and environmental aspects from ethical and civilizational perspectives, producing morally responsible and professional graduates.


#### Key Topics Learned

* Introduction to Ethics and Civilizations
* Concepts of Individual, Society, State, and Global Citizenship
* Classical and Modern Ethical Theories
* Ethics in Various Civilizations:
  * Islamic Civilization
  * Malay Civilization
  * Chinese Civilization
  * Indian Civilization
  * Western Civilization
* Social Cohesion and National Integration
* Federal Constitution as a Pillar of Ethics and Civilization in Malaysia
* Contemporary Issues:
  * Politics
  * Economy
  * Social Issues
  * Culture
  * Environment
* Social Responsibility:
  * Individual Responsibility
  * Professional Ethics
  * Corporate Responsibility
  * Environmental Responsibility
* Sustainable Development Goals (SDG) and Global Challenges


#### Skills Acquired

* Critical Thinking and Ethical Analysis
* Understanding of Civilizational Diversity
* Communication and Presentation Skills
* Essay Writing and Argument Development
* Teamwork and Collaboration
* Awareness of Social Responsibility and National Unity


#### Activities & Learning Methods

* Group Presentations
* Creative Video Production
* Essay Writing on Contemporary Issues
* Games and Interactive Learning Activities
