### SECP2613 – System Analysis and Design

This course focuses on analyzing system requirements and designing information systems using structured methodologies and UML modeling techniques. It includes system planning, requirement analysis, and design documentation.

#### Key Topics Learned

* Systems Analysis Fundamentals
* Project Planning and Feasibility Study
* Data Flow Diagram (DFD)
* Requirement Gathering Techniques
* Use Case Diagram (UML)
* System Design (Input, Output, Interface Design)
* System Development Lifecycle

#### Skills Acquired

* System Analysis and Design
* UML Diagramming
* Requirement Analysis
* Project Management
* Technical Documentation

#### Technologies Used

* UML Tools (Draw.io)
