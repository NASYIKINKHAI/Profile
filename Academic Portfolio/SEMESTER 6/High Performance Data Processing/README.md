### SECP3133 – High Performance Data Processing

This course introduces cloud computing, HPC concepts, and large-scale data processing techniques. It focuses on distributed systems and modern computing frameworks for big data applications.

#### Key Topics Learned
- Cloud Computing Concepts
- High Performance Computing (HPC)
- Distributed Data Processing
- Big Data Systems
- Parallel Processing
- Scalable Data Architecture

#### Skills Acquired
- Cloud & HPC Understanding
- Distributed Computing Skills
- Data Processing at Scale
- Analytical Problem Solving
- System Optimization
