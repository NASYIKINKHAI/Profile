### SECP3106 – Application Development (WBL)

This Work-Based Learning course focuses on developing real-world applications through industry exposure and practical implementation. Students apply software development concepts in a professional environment.

#### Key Topics Learned
- Application Development Lifecycle
- System Design & Implementation
- Software Development Practices
- Industry-based Requirements
- Testing & Deployment

#### Skills Acquired
- Full-stack Application Development
- Software Engineering Practices
- Problem Solving in Real Systems
- Team Collaboration
- Industry Communication Skills
